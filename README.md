# My Interactive AI Portfolio

A dynamic, web-based professional portfolio featuring "Edith," a custom AI assistant powered by the Google Gemini API. 

## 🚀 Features
* **AI Chatbot (Edith):** Answers recruiter questions about projects, skills, and background.
* **Secure Architecture:** Built with a Python/Flask backend to secure API keys.
* **Interactive UI:** Custom HTML/CSS/JS frontend with particle physics canvas backgrounds.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript
* **Backend:** Python, Flask
* **AI Integration:** Google Gemini API

## 🛠️ Local Setup
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Create a `.env` file and add your `GEMINI_API_KEY`.
4. Run the server: `python app.py`
